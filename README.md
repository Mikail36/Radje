# Development Setup

## Prerequisites

* Node.js (Latest LTS Version Recommended)
* npm Package Manager

## Installation

```bash
# Clone the repository
git clone <YOUR_REPOSITORY_URL>

# Navigate to the project folder
cd <PROJECT_NAME>

# Install dependencies
npm install

# Start development server
npm run dev
```
